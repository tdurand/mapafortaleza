h3. Mapa de onibus de Fortaleza

You can read an article on my blog about the story of this project (and a bit of architecture): 
http://tdurand.tumblr.com/post/31393552608/how-i-created-a-bus-map-of-fortaleza-brazil

*NB: The data are not automatically updated.*

Application for visualizing bus lines of Fortaleza:

* The data are from "Etufor":http://www.fortaleza.ce.gov.br/etufor/index.php?option=com_content&task=view&id=278 .
* The application uses Google Fusion Table to manipulate data
* Thanks to : "wolpil.de":http://www.wolfpil.de/v3/arrow-heads.html for the code of arrows for polylines
